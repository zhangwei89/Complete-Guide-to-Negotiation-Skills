# 第 20 章 房产与汽车交易

## 开篇：黄小玲买二手房被骗了 30 万

黄小玲两年前买了一套二手房，卖家开价 480 万。她实地看了 3 次，每次都听中介说"这套房子很多人想要"。

她没做任何议价。她直接给卖家报价 480 万，卖家说"考虑一下"，第二天中介打电话说"卖家同意了，但要你再付 5 万中介费"。

她付了。

后来她才知道，这套房子当时的市场价是 410 万。她多付了 70 万。

她后来请教了一个做房产的亲戚，亲戚说：

- 480 万 的报价里，至少有 30-40 万 的议价空间。
- 二手房通常还有 5-10 万 的可议空间（甚至 20 万）。
- 你不议价，卖家就赚你的差价。

她后来写下："买房子是人生最大的一笔交易，但大多数人包括我，完全没有议价能力。"

这一章我们解决房产 / 汽车交易。

## 一、买房议价：5 步法

### 步骤 1：市场调研

买房前，做市场调研：

- 查 3 个月同小区成交价（链家 / 贝壳 / 我爱我家）。
- 看近期挂牌价。
- 看卖家是否急卖（挂牌时间长 = 急卖）。

实战工具：

- 链家 / 贝壳 APP。
- 我爱我家 APP。
- 实地走访中介。

### 步骤 2：算清底价

- 你的预算上限 = 总价 + 税费 + 中介费 + 装修费 + 家具家电。
- 你的目标价 = 市场均价 - 5%。
- 你的底线 = 卖方报价 - 10% - 20%。

### 步骤 3：试探卖家

试探卖家的态度：

- 卖家是否急卖？（挂牌时间 / 看房频率）
- 卖家为什么要卖？（换房 / 资金 / 移民）
- 卖家的杠杆有多高？（银行欠款多少）

试探方法：

- "您这套房子挂了多久了？"
- "您卖了买哪里？"
- "这套房子的优点是什么？缺点呢？"，让卖家说出缺点。

### 步骤 4：报价

报价策略：

- 不要单点报价，报 3 档（首付高 / 总价中 / 付款快）。
- 第一报比市场价低 10%。
- 让步幅度递减。

### 步骤 5：谈判

谈判 5 个注意：

1. 不要立刻接受卖方的报价，你至少让 1 次。
2. 用卖方着急点（"卖家是不是急卖？"）压价。
3. 用房子缺点压价，你已经发现的问题。
4. 用同小区成交价压价，"同小区 3 个月内成交均价是 X"。
5. 用付款速度换价，"我可以 1 周内付首付，希望总价能再降 X"。

### 议价的常见套路

卖家 / 中介会用以下套路，你要识破：

- 套路 1：制造竞争（"还有另一家在谈"，可能假）。
- 套路 2：最后通牒（"您今天不签就不行了"，通常假）。
- 套路 3：信息不对称（"这套是稀缺户型"，可能是真的，但要验证）。
- 套路 4：情绪压力（"您看孩子多喜欢这套"，你要保持冷静）。
- 套路 5：找"权威"（"我们老板说这价不能动"，可能是中介配合卖家的策略）。

## 二、租房谈判：4 步法

### 步骤 1：市场调研

- 同小区 / 同户型的租金（链家 / 贝壳 / 58）。
- 中介费通常是多少（月租 35-50%）。

### 步骤 2：准备

- 你的预算。
- 你的入住时间。
- 你的合同期长（长合同 = 议价空间大）。

### 步骤 3：谈判

议价的 5 个点：

1. 月租本身（市场价 - 5%）。
2. 中介费（通常是月租 35-50%，可以谈到 25-30%）。
3. 押金（押一付一 / 押一付三 / 押二付一）。
4. 入住时间（提前入住 / 晚入住）。
5. 房屋维修（合同里写明哪些费用房东承担）。

### 步骤 4：签约

合同细节：

- 房屋现状（家具 / 家电 / 水电 / 网络）。
- 维修责任（哪些房东修，哪些你修）。
- 退租条款（提前多久通知 / 押金退还条件）。
- 中介费（具体金额）。

## 三、二手车交易：5 步法

### 步骤 1：调研

- 同款车 3 个月内成交价（瓜子 / 人人车 / 优信二手）。
- 车辆历史（4S 店保养记录 / 维修记录 / 出险记录）。
- 车辆公里数 / 上牌时间。

### 步骤 2：评估车况

- 外观（划痕 / 凹陷）。
- 内饰（座椅 / 仪表 / 中控）。
- 机械（发动机 / 变速箱 / 底盘）。
- 试驾（启动 / 加速 / 刹车 / 异响）。

### 步骤 3：议价

二手车议价空间通常是 5-15%。

议价的 5 个点：

1. 车辆公里数（每超 1 万公里减 X 元）。
2. 车辆年限（每超 1 年减 X 元）。
3. 车辆缺陷（每处减 X 元）。
4. 保养记录（不全减 X 元）。
5. 出险记录（有事故减 X 元）。

### 步骤 4：检测

议价后，做一次专业检测：

- 4S 店检测（500-1000 元）。
- 第三方检测机构（300-500 元）。

检测出问题，再议价。

### 步骤 5：过户

过户细节：

- 车辆过户手续。
- 保险过户。
- 车牌过户（如需要）。
- 车辆违章查询（过户前）。

### 二手车议价常见套路

- 套路 1：调表（修改公里数）。
- 套路 2：事故车当非事故车卖。
- 套路 3：泡水车当非泡水车卖。
- 套路 4：4S 店保养记录不全，可能保养差。
- 套路 5：临时加价，"刚有人出 X 元"。

## 四、新车优惠谈判：5 步法

### 步骤 1：行情调研

- 4S 店官方优惠。
- 网络优惠（汽车之家 / 易车 / 懂车帝）。
- 二手车平台报价（用来议价）。

### 步骤 2：选定车型

选定 2-3 个车型（不要只用 1 个，否则没有议价空间）。

### 步骤 3：议价

新车议价的 5 个点：

1. 现金优惠（直接降价）。
2. 礼包（脚垫 / 贴膜 / 行车记录仪等）。
3. 保养套餐（几次免费保养）。
4. 保险（4S 店保险 / 外面保险）。
5. 置换补贴（如果有旧车）。

### 步骤 4：金融方案

如果贷款，议价：

- 利率（厂家金融 vs 银行）。
- 手续费。
- GPS 费（如有）。
- 上牌费（如有）。

### 步骤 5：合同

合同细节：

- 价格（写明现金优惠 + 礼包）。
- 配置（写明具体配置）。
- 交车时间。
- 违约责任。

### 新车议价常见套路

- 套路 1：4S 店报"底价"，其实还有空间。
- 套路 2：厂家金融利率低，但有手续费。
- 套路 3：保险强制 4S 店，可以外面买。
- 套路 4："今天不签就没这价"，可能假。

## 五、售后服务争取：5 个方法

### 方法 1：免费保养

谈判时，谈"几次免费保养"（通常 1-3 次）。

### 方法 2：延保

- 官方延保（厂家提供的延长保修期）。
- 谈判免费或低价延保。

### 方法 3：保养套餐

预付保养套餐，价格更低。

### 方法 4：道路救援

- 免费道路救援（电池救援 / 拖车）。
- 谈判加入。

### 方法 5：置换优惠

如果未来打算置换，谈"置换优惠 +X 元"。
