# 第 13 章 心理博弈

## 开篇：陈伟和张总的"心理战"

陈伟第一次见投资人张总前，他以为谈判是"谈条件"，谈估值、谈股份、谈董事会席位。

但他经历了 90 分钟会议后，他意识到，谈判的 80% 是"心理"，20% 是"条件"。

张总在会议开头就在做心理铺垫，

1. 张总微笑着说："陈总，我们之前也投过类似的赛道，XX 项目，他们当时估值 8000 万，但一年后破产了。所以我对这个赛道比较谨慎。"（用权威 + 锚定 + 损失厌恶）

2. 张总叹了口气："陈总，这个时间点行业普遍在降估值。我们最近看的几个项目平均降了 30%。"（用一致性 / 锚定）

3. 张总突然"关心"陈伟："陈总，你今年多大了？我看你创业经历挺有意思。"（建立亲和 + 信息收集）

4. 张总最后说："陈总，我看您的方案完整度还不够，我们更愿意支持另一个更成熟的创始人。"（制造竞争 + 损失厌恶）

陈伟完全被张总的心理策略压制，他没能坚持自己的估值，最后答应了不合理的条件。

他后来复盘：张总用了 5 种心理原理，锚定、权威、损失厌恶、稀缺、制造竞争。他每一条都被击中。

这一章我们就拆解这些心理原理。

---

## 一、锚定效应（再次深入）

第 2 章我们讲过锚定效应，这一节深入到"实战博弈"层。

### 锚定的 3 个层次

层次 1：数字锚

最直接的锚，具体的数字。

实战：

- 你的报价是 380 万。客户第一次回价是 220 万，这就是数字锚。
- 你的目标是 6 月上线。客户问"为什么不是 8 月"，这是时间锚。

层次 2：范围锚

比数字锚更精妙，给一个"范围"而不是具体数字。

实战：

- 你的报价是"350-450 万"。客户回价"200-250 万"。
- 这个锚比数字锚更稳，对方很难突破。

层次 3：参照锚

最隐蔽的锚，用一个"参照"作为锚。

实战：

- "行业惯例是 30% 利润率，您这个项目凭什么要 50%？"，这是用"行业惯例"做参照锚。
- "我们通常的做法是 XXX"，用"通常"做锚。

### 反锚定的 5 个方法

方法 1：忽略第一个锚

不要在心理上接受对方的第一个锚。把第一个报价当作"开场"，不是"终局"。

方法 2：抛出反锚

"您说 200 万，但根据我们的成本，350 万才是合理的起点。"，抛出你的反锚。

方法 3：扩大锚的范围

不说 380 万，说"350-420 万，具体取决于交付时间"，把锚变成"范围"，让对方难以聚焦。

方法 4：用参照反锚

"据我所知，类似项目市场报价在 380-450 万，您这个 200 万的还价太低了。"。

方法 5：沉默反锚

对方报 200 万。你不立刻回应。等 5 秒。对方会有压力。这个沉默本身就是反锚。

## 二、损失厌恶（再次深入）

### 损失厌恶的 3 类应用

应用 1：把收益转化为损失

不要告诉对方"您会得到 X"，告诉对方"您不这么做会失去 X"。

实战：

- 不要说"我们的方案能让您节省 8 万"。
- 说"如果您不选择我们的方案，按您现在的成本，您每年会多支出 100 万"。

应用 2：放大对方已有的"潜在损失"

对方已经有的痛点，你帮他"放大"。

实战：

- "您刚才提到库存周转慢，我估算这种慢一年让您的资金占用增加 600 万。这是您现在就在承受的损失"。

应用 3：让对方"先失去"再"拿到"

先让步一点，让对方感觉"失去"你的让步，再恢复。

实战：

- 你让步 5% — 对方开心。
- 你立刻说"但有个条件"，对方感觉"失去"5%，你想恢复，需要交换。

### 反损失厌恶的 3 个方法

方法 1：识别对方的损失语言

"如果您不这么做……""如果不选我们的方案……"，对方在用损失语言。

方法 2：重新框架为收益

"我能理解您的不安全感，但换个角度想，您选了我们的方案每年能多 100 万。"。

方法 3：把你的让步"重新框架"

你让步 5%，你可以说"我让步 5% 是因为我们重视您的长期合作，这是我们的诚意"，把让步变成收益框架。

## 三、稀缺效应

### 稀缺的 4 类表现

表现 1：限时

"这个优惠到本周五截止。"。

表现 2：限量

"这个产品只剩最后 5 个。"。

表现 3：限地

"这个方案我们只和 3 家客户做。"。

表现 4：限人

"这个机会我特别给您的，别人没有。"。

### 稀缺的应用方法

实战应用：

- 在合适的时机提供限时 / 限量 / 限地 / 限人。
- 关键是"真的稀缺"，假的稀缺被识破就失效。

### 反稀缺的 3 个方法

方法 1：检验稀缺是否真实

"这个方案您可以一直提供吗？还是只在特殊时期？"。

方法 2：识破伪稀缺

"您这个'稀缺'是为了让我快速决策吗？"，直接问。

方法 3：利用稀缺反压

"我理解您的稀缺，我也希望您能尽快。"，用对方的稀缺反压。

---

## 四、从众心理

### 从众的 3 类表现

表现 1：行业内从众

"我们同行 80% 都选了这个。"。

表现 2：客户从众

"我们已经有 100 个客户选了这个方案。"。

表现 3：权威从众

"XX 大学的研究显示……""XX 协会的标准是……"。

### 从众的应用方法

实战应用：

- 提到行业惯例。
- 提到客户案例。
- 提到权威研究。

关键：每个从众要有具体数字 + 具体出处。

### 反从众的 3 个方法

方法 1：识别从众心理

"您说'80% 的同行都做'，具体是哪些同行？"，如果对方答不上来，是策略。

方法 2：找反例

"我听说 XX 同行用的不一样的方式，效果也很好。"。

方法 3：强调差异化

"我们的方案可能和 80% 的同行不一样，但正是这样我们才能给您独特价值。"。

## 五、承诺一致性

### 承诺的 4 类层级

层级 1：弱承诺

"我可以考虑"，最弱的承诺。

层级 2：偏好承诺

"我倾向于……"，稍强的承诺。

层级 3：行动承诺

"我会做……"，更强的承诺。

层级 4：公开承诺

"我已经公开说……"，最强的承诺。

### 利用承诺的 3 个方法

方法 1：从小承诺到大承诺

先让对方做小承诺（"我们先见一次"）→ 再让对方做大承诺（"我们可以试试"）。

方法 2：把承诺说出来

"您刚才说您比较看重快速上线，那您愿不愿意签这个速度条款？"。

方法 3：利用对方的过去承诺

"您上次说……您觉得这次有什么不一样吗？"。

### 反承诺的 3 个方法

方法 1：不要在小承诺上让步

你的小承诺（"我会考虑"）如果不收回，你就要真的考虑。

方法 2：明确你还没承诺

"我刚才说的是'考虑'，不是'同意'。"。

方法 3：让对方公开承诺

"您能不能在公司内公开说……"，把对方的承诺加重。

## 六、心理博弈的边界

心理博弈是工具，不是目的。博弈要在"让对方觉得合理"的范围内。

边界 1：让对方"事后觉得这是理性的"

如果你用博弈让对方做了让步，事后对方觉得"我当时是被骗了"，关系破裂。

边界 2：不要重复使用同一个博弈

对方的同一博弈你识破一次，下次同样的博弈对对方失效。

边界 3：博弈要有"退出空间"

让对方有台阶下，博弈不能用得太绝。

边界 4：博弈要看长期

如果博弈让你赢一次但失去长期客户，博弈失败。

## 七、一个正面案例：王琦用"心理博弈"挽救订单

王琦做销售，遇到一个特别难缠的客户，陈总。

陈总用了很多策略：

- "我们接触的另一家只要 80 万"（制造竞争）。
- "我今天就要签"（最后通牒）。
- "我们行业里的标准价是 100 万"（从众）。
- "我老板说不行"（权威）。

王琦没有被击穿，她用了反策略：

- "我能理解陈总希望快速确定，我这边需要 24 小时给您答复"（用时间反压）。
- "您能不能详细告诉我那家公司是什么方案？"（要求细节）。
- "据我了解，类似项目的标准价是 120-150 万"（用反从众）。

最后订单以 130 万成交。陈总后来说"你是这么多销售里最有策略的"。

### 这个案例的关键：

- 每个心理策略都有反策略。
- 反复使用反策略，对方策略失效。
- 长期关系的关键是"让对方觉得这是理性的"。

## 八、一个反面案例：陈伟被"权威 + 损失厌恶"击穿

陈伟第一次见张总时，完全被张总的心理策略击穿。

张总：

- "我们之前投的类似项目破产了，我对这个赛道谨慎"（权威 + 损失厌恶）。
- "行业普遍降估值 30%"（从众）。
- "我老板支持另一个创始人"（稀缺 + 制造竞争）。

陈伟：

- 完全接了所有锚。
- 完全接受了损失框架。
- 没有识别从众是真是假。
- 没有识别竞争者是真的还是假的。

他最后接受了 5000 万估值，他原计划是 7500 万。

### 这个案例的反面教训：

- 心理博弈的每一个原理都可能是"陷阱"。
- 你需要识别 + 应对 + 不立刻反应。

## 九、识别对方在用什么心理原理的 5 个观察点

观察 1：对方的"信息呈现方式"

对方在用数字 / 数据 / 案例？，他可能在用锚定 / 从众 / 权威。

观察 2：对方的"语言结构"

"您会失去……""如果不……"，他在用损失厌恶 / 稀缺。

观察 3：对方的"时间压力"

"今天就要……""来不及了"，他在用稀缺 / 时间压力。

观察 4：对方的"情绪变化"

他突然情绪激动 / 突然冷淡，他在用情绪施压。

观察 5：对方的"承诺"强度

"我会考虑""我会做""我公开说"，他在用承诺一致性。

## 章末行动清单

下次谈判前，准备好 5 个心理原理的反策略：

1. 锚定：用反锚 / 范围锚 / 沉默反锚。
2. 损失厌恶：用收益框架 / 重新框架。
3. 稀缺：检验真假 / 识破伪稀缺 / 反压。
4. 从众：识别从众 / 找反例 / 强调差异化。
5. 承诺一致性：不立刻做小承诺 / 明确还没承诺 / 让对方做更大承诺。

## 实战演练

**演练 1**：复盘你最近一次谈判。对方用了哪个心理原理？你被击中了吗？你下次怎么识破？

**演练 2**：下次谈判前，列出对方最可能用的 3 个心理原理。每个原理列出：
- 你如何识别（识别信号）。
- 你如何反制（具体话术）。
- 你的底线（不让步到哪里）。

**演练 3**：找一个你信任的朋友，模拟"心理博弈"。你扮演客户用 3 个心理原理攻击；他练习识别 + 反制。演练 2 次后，他会明显提升。
