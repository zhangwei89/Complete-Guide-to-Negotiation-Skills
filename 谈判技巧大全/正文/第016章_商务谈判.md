# 第 16 章 商务谈判

## 开篇：陈伟签下第一笔战略合作

陈伟的 SaaS 公司做了一年，他第一次有机会和一家大型企业签战略合作。

对方是一家年营收 50 亿的上市公司，IT 总监张总主动找陈伟谈。战略合作 5 年期，预估总价值 2000 万。

陈伟这次做了完全不同的准备：

1. **准备阶段**：他花了一周时间画清楚对方的决策链（张总 + 张总的老板 + 法务 + 财务）。他和张总私下聊过 3 次，了解了对方最关心的不是价格，是"数据安全 + 长期稳定"。

2. **报价阶段**：他设计了 3 档报价（基础 350 万 / 推荐 480 万 / 战略 700 万）。战略档包含 5 年期独家 + 数据本地化 + 高级安全 + 季度业务复盘。

3. **让步阶段**：他没有单独让步，每次让步都换对方的承诺。比如让 30 万换 5 年期承诺，让 50 万换季度业务复盘授权。

4. **节奏阶段**：他把整个谈判分成 4 次会议。每次会议只解决 1-2 个关键议题。每次会议结束都发一份纪要，确保对方内部对齐。

5. **关系阶段**：每次会议前他都和张总单独吃顿饭，建立私人关系。

3 个月后，他签下这份战略合作，480 万 / 年，5 年期，对外还能公开作为案例。

这次成功不是偶然，是商务谈判的系统方法。

这一章我们就拆解商务谈判的完整流程。

## 一、商务谈判的 5 个特征

商务谈判和日常谈判不一样。它有 5 个独特特征。

### 特征 1：金额大

商务谈判通常涉及 100 万到几千万的金额。

影响：

- 让步幅度大。
- 决策时间长。
- 决策人多。

### 特征 2：决策链长

商务谈判的决策人通常不是 1 个人，是 3-7 个人。

影响：

- 谈判前必须画出对方决策链。
- 接触每个决策人。

### 特征 3：周期长

商务谈判通常 1-6 个月。

影响：

- 不能一蹴而就。
- 必须做节奏管理。
- 必须做关系维护。

### 特征 4：条款多

商务谈判的合同通常 30-100 页条款。

影响：

- 必须准备条款清单。
- 不能放过细节。
- 必须有法律团队支持。

### 特征 5：关系重要

商务谈判不是一次性，5-10 年的合作常见。

影响：

- 关系是前提。
- 不能做"杀鸡取卵"的让步。
- 必须考虑长期。

## 二、商务谈判的完整流程：6 阶段

### 阶段 1：准备（1-2 周）

核心动作：

- 收集对方信息。
- 画出对方决策链。
- 准备 3 档报价。
- 准备让步路径。
- 准备 BATNA。
- 内部对齐（我方）。

产出：

- 准备清单（附录 A）。
- 3 档报价方案。
- 让步矩阵。

### 阶段 2：开局（30-60 分钟）

核心动作：

- 双方高层见面。
- 建立第一印象。
- 确认合作框架。
- 确认谈判议程。

不要做：

- 不要立刻报价。
- 不要立刻讨论细节。
- 不要在开局就让步。

### 阶段 3：探测（多次会议）

核心动作：

- 询问对方需求。
- 询问对方预算。
- 询问对方决策流程。
- 探底，找到对方的核心利益。

方法：

- 5 类提问（开 / 闭 / 引 / 假 / 追）。
- 倾听 3 层（事实 / 情绪 / 需求）。
- 主动探底（"如果我们再降 5%，您能在哪个条款上让步？"）。

### 阶段 4：谈判（多次会议）

核心动作：

- 报 3 档报价。
- 应对压价 / 涨价。
- 让步必换。
- 设计条款包。

方法：

- 多方案报价（good-better-best）。
- 让步必换。
- 条件交换。
- 应对策略（虚张声势 / 最后通牒 / 红脸白脸）。

### 阶段 5：收尾（1-2 周）

核心动作：

- 确认所有共识。
- 写详细合同。
- 内部审批。
- 法务审核。
- 签字。

不要做：

- 不要在最后一刻让步太大。
- 不要忽略未说清楚的条款。
- 不要急着签。

### 阶段 6：维护（长期）

核心动作：

- 兑现承诺。
- 季度复盘。
- 主动反馈。
- 关系维护。

## 三、商务谈判的 4 大策略

### 策略 1：先合作后博弈

商务谈判最有效的策略是先合作后博弈。

错误：把谈判当竞争，结果是零和博弈。

正确：先建立合作氛围，再在分歧点上博弈。

具体做法：

- 前 30% 的会议不谈价格，只聊合作可能。
- 中段 40% 的会议谈方案和价值。
- 后段 30% 的会议谈价格和条款。

### 策略 2：把蛋糕做大

商务谈判不是单纯分蛋糕，是把蛋糕做大。

错误：把谈判当分配，双方都在抢。

正确：找出双方都能接受的"做蛋糕"方式。

具体做法：

- 加项不加价（增加服务但不增加价格）。
- 用未来合作换现在利益。
- 用非财务利益换财务利益。

### 策略 3：多方案报价

商务谈判必须用多方案报价，不能单点报价。

错误：报 380 万，客户直接说"贵"，你没有空间。

正确：报 3 档，客户选 1 个，你没有让步压力。

### 策略 4：长期主义

商务谈判必须有长期视角。

错误：为单次合作最大化利益，失去未来。

正确：为 5-10 年合作建立信任，单次让步是合理的。

## 四、产品采购场景

产品采购是商务谈判里最常见的场景。

### 采购谈判的 5 步流程

步骤 1：明确需求

- 我需要什么产品 / 服务？
- 规格 / 数量 / 时间 / 质量要求是什么？

步骤 2：寻找供应商

- 至少 3 家备选。
- 评估每家的能力 / 价格 / 风险。

步骤 3：报价比价

- 让每家报 3 档。
- 比较各家的"总获取成本"（不只是单价）。

步骤 4：谈判

- 用本书所有策略。

步骤 5：签约 + 维护

- 详细合同。
- 长期维护。

### 采购谈判的 3 个特殊点

特殊点 1：总获取成本（TCO）

不要只看单价，要看总获取成本。

TCO = 单价 + 运输 + 维护 + 培训 + 风险。

特殊点 2：质量 > 价格

采购最常见的错，只压价格，但质量不行。

错误：选最便宜的，质量差，返工，成本反而高。

正确：选性价比最高的，质量 + 价格综合考虑。

特殊点 3：长期合作

采购不是一次，是 5-10 年的合作。

短期压价会损失长期合作，得不偿失。

---

## 五、销售合作场景

销售合作是商务谈判的另一面，卖方视角。

### 销售谈判的 5 步流程

步骤 1：识别客户

- 谁是我的客户？
- 客户的痛点是什么？

步骤 2：建立信任

- 用案例 / 数据建立信任。
- 用 3 步信任建立（能力 / 善意 / 透明）。

步骤 3：呈现方案

- 报 3 档方案。
- 用 4 步表达（立场 / 支撑 / 例子 / 行动）。

步骤 4：谈判

- 应对压价。
- 让步必换。
- 处理异议。

步骤 5：签约 + 维护

- 详细合同。
- 长期维护。

### 销售谈判的 3 个特殊点

特殊点 1：价值先于价格

销售最常见的错，先谈价格。

正确：先谈价值，让客户先认可价值，再谈价格。

特殊点 2：异议处理

销售谈判里客户会提很多异议。每个异议都是机会。

异议处理 4 步：

- 倾听异议。
- 复述异议。
- 探底异议（"您具体担心什么？"）。
- 回应异议。

特殊点 3：长期客户

销售不是一次，是长期客户关系。

不要为单次销售最大化，为长期合作建立信任。

---

## 六、合同签订场景

合同签订是商务谈判的"终点"，也是长期关系的起点。

### 合同谈判的 5 个要点

要点 1：条款清单

不要只看总价，要看所有条款。

条款清单示例：

- 价格 / 数量 / 规格。
- 付款方式 / 付款节点。
- 交付时间 / 交付方式。
- 质量标准 / 验收方式。
- 售后 / 维护 / 培训。
- 保密 / 知识产权。
- 违约 / 终止。
- 争议解决。

要点 2：风险条款

每个条款必须有风险条款。

示例：

- 质量不达标怎么办？
- 延期交付怎么办？
- 付款延期怎么办？

要点 3：争议解决条款

必须有明确的争议解决方式。

示例：

- 协商 → 调解 → 仲裁 → 诉讼。
- 仲裁地 / 仲裁机构。

要点 4：变更条款

合同执行中可能有变更，必须有变更条款。

示例：

- 双方协商变更。
- 书面变更才有效。
- 变更的范围和方式。

要点 5：终止条款

合同必须有终止条款，保护双方。

示例：

- 双方协商终止。
- 单方违约终止。
- 不可抗力终止。

### 合同谈判的常见错误

错误 1：只关注价格

不看条款，只看总价，后期被条款坑。

错误 2：默认对方条款

直接用对方合同，条款对对方有利，你吃亏。

错误 3：不看违约条款

不仔细看违约条款，出问题没保护。

错误 4：不看终止条款

不仔细看终止条款，合作失败时退出困难。

### 合同谈判的实战建议

- 合同条款必须逐条谈，不要批量接受。
- 高风险条款（如保密 / 知识产权 / 违约）必须有专业律师审。
- 任何口头承诺必须写到合同里，口头不算。

## 七、项目合作场景

项目合作是商务谈判里最复杂的，多个利益方 + 多个目标。

### 项目谈判的 5 个核心点

核心点 1：双方目标对齐

项目谈判最容易失败的是，双方目标不一致。

正确做法：先把双方目标对齐，再谈条款。

核心点 2：决策链识别

项目合作通常有多个决策人，必须识别所有。

核心点 3：里程碑 + 节点

项目合作必须分阶段，每个阶段有里程碑。

里程碑示例：

- M1：方案确定。
- M2：开发完成。
- M3：上线完成。
- M4：验收完成。
- M5：维护开始。

每个里程碑有付款节点。

核心点 4：风险共担

项目合作有风险，必须风险共担。

风险分担方式：

- 双方投入比例。
- 双方承担的责任。
- 双方承担的成本。

核心点 5：退出条款

项目合作必须有退出条款。

退出条款示例：

- 什么情况可以退出。
- 退出后的资产归属。
- 退出后的付款安排。

## 八、长期战略合作场景

长期战略合作是商务谈判的"最高级"形式，通常 5-10 年期。

### 战略合作的 4 个特征

特征 1：金额大

战略合作通常年度 1000 万以上。

特征 2：周期长

5-10 年是常态。

特征 3：双方投入大

战略合作通常需要双方投入（团队 / 资源 / 资金）。

特征 4：协同效应

战略合作的关键是"协同"，1+1 > 2。

### 战略合作的 5 个要素

要素 1：愿景一致

战略合作必须有共同的愿景，不是"赚钱"，是"共同的方向"。

要素 2：互信

战略合作的基础是互信，信任需要长期建立。

要素 3：规则清晰

战略合作必须有清晰的规则，避免冲突。

要素 4：退出机制

战略合作必须有退出机制，不是分手，是"调整"。

要素 5：长期主义

战略合作必须是长期主义，不为短期利益牺牲长期。

### 战略合作谈判的 6 个注意

注意 1：分阶段签

战略合作不要"一次性签 10 年"，分阶段签。

阶段示例：

- 第 1 期：2 年，验证合作。
- 第 2 期：3 年，扩展合作。
- 第 3 期：5 年，战略合作。

注意 2：明确 KPI

每个阶段必须有明确的 KPI，避免合作无效果。

注意 3：定期复盘

每季度 / 每年复盘，调整合作。

注意 4：升级机制

战略合作必须有升级机制，出问题能升级处理。

注意 5：风险共担

战略合作必须有风险共担，不能只一方承担。

注意 6：知识产权

战略合作的知识产权必须明确，避免争议。

## 九、商务谈判的常见错误

### 错误 1：只看价格

只看价格不看条款，后期被条款坑。

### 错误 2：不看决策链

不知道对方的决策链，接触错的人，浪费时间。

### 错误 3：不为长期着想

为单次合作最大化，失去长期合作，得不偿失。

### 错误 4：不做准备

不做准备，被对方主导，输掉谈判。

### 错误 5：单点报价

单点报价，没有空间，被压价。

### 错误 6：不写备忘

谈判不写备忘，后期对方不承认，争议。

### 错误 7：急着签

急着签，错过关键条款，后期麻烦。

## 十、章末行动清单

下次商务谈判前，提醒自己：

1. 我画出对方决策链（至少 3 个人）。
2. 我设计 3 档报价（good-better-best）。
3. 我准备让步路径（4-5 次，幅度递减）。
4. 我准备 BATNA（至少 3 个）。
5. 我用"先合作后博弈"的策略。
6. 我把谈判分成 6 阶段（准备 / 开局 / 探测 / 谈判 / 收尾 / 维护）。
7. 合同条款逐条谈，不批量接受。
8. 谈判写备忘，每次会议结束发纪要。

## 实战演练

**演练 1**：为你下一次商务合作，画出对方决策链。识别 3-5 个决策人，每个决策人写出"他的核心利益"。

**演练 2**：为这个合作设计 3 档报价。每档包含：价格、服务、时间、付款方式、特殊承诺。

**演练 3**：列出你商务谈判里的 3 类常见错误 + 你下次怎么避免。
