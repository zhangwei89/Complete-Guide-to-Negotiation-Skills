# 第 17 章 职场谈判

## 开篇：苏清的晋升对话

苏清做运营经理 5 年，她去年主动找老板谈了一次晋升。

她之前的方法都是"等老板主动"，结果是连续 3 年没晋升。

她这次学了一种新方法，"主动方案式争取"。

她做了 5 件事：

1. **准备阶段**：她提前 2 周整理了她过去 5 年的成绩清单（12 项量化结果 + 5 个关键项目 + 3 个团队建设成果）。

2. **价值重构**：她不直接说"我要晋升"，她说"老板，我想和您聊聊公司 Q3 的运营规划，我也想贡献更多"。

3. **场景设计**：她约老板在周四下午 4 点的非正式场合（小会议室 + 两杯茶），不是周一早上（老板压力大）。

4. **方案呈现**：她用金字塔原理呈现，结论先说、3 点支撑、1 个案例、1 个行动。

5. **复述 + 沉默**：老板问她"你觉得你的贡献是什么？"她不立刻回答，她沉默 3 秒，然后说"老板，您觉得呢？"，这是给对方表达的机会。

那次对话后 2 周，老板给了她晋升，而且加了 25% 工资。

她后来说："我之前失败是因为我在'等'老板给我机会，成功是因为我开始'主动'创造机会。"

这一章我们就解决职场谈判。

---

## 一、职场谈判的 5 个独特点

### 独特点 1：长期关系

职场是长期关系，5 年、10 年、20 年。

影响：

- 不能用"一次性"策略。
- 必须维护长期关系。
- 不能为短期利益牺牲长期。

### 独特点 2：权力不对等

职场里有明确的层级，权力不对等。

影响：

- 弱势方（员工）有特定的谈判策略。
- 强势方（老板）也有特定的限制（不能太明显）。

### 独特点 3：评价体系

职场里有评价体系（KPI / OKR / 360 评估）。

影响：

- 谈判要看评价体系。
- 你的"贡献"必须可量化。
- 评价好的谈判效果更好。

### 独特点 4：分配机制

职场里的资源（薪资 / 晋升 / 资源）是分配的，不是赢得的。

影响：

- 你不是从老板那里"赢"，是从老板那里"分配"。
- 你要让老板觉得"分配给你"对他也有好处。

### 独特点 5：多重关系

职场里你和老板的关系不只是"工作"，还有信任 / 期待 / 利益 / 情感。

影响：

- 不能纯理性谈判。
- 必须考虑情感 / 信任。

## 二、薪资谈判：5 步流程

### 步骤 1：自我评估

回答以下问题：

- 我过去 1 年的核心成果是什么？（至少 5 个量化结果）
- 我在市场上值多少钱？（同行业 / 同岗位 / 同经验）
- 我的 BATNA 是什么？（如果谈崩我能去哪）

### 步骤 2：研究市场

- 同行业同岗位薪资范围。
- 同经验同事的薪资范围。
- 公司内部薪资范围（如果你能查到）。

### 步骤 3：定目标

- 理想值：你希望涨多少。
- 可接受值：你愿意接受的最低涨多少。
- 底线值：低于多少你不接受。

### 步骤 4：场景设计

- 时间：选老板相对放松的时间（不要周一早上 / 季度末）。
- 地点：选非正式场合（小会议室 / 咖啡馆）。
- 氛围：先聊合作再聊薪资。
- 准备：用 4 步结构呈现（立场 / 支撑 / 例子 / 行动）。

### 步骤 5：谈

- 开场：用老板关心的话题开场（"老板，我想和您聊聊 Q3 团队规划"）。
- 价值呈现：用 3 个量化结果支撑你的贡献。
- 期望呈现：明确你的期望（"我希望薪资能到 X"）。
- 倾听：听老板的反应。
- 谈判：必要时让步。
- 收尾：确认共识，写备忘。

### 薪资谈判的 5 个注意

注意 1：不威胁离职

"如果您不加薪我就离职"，会让老板防御。

正确："我希望在公司长期发展，所以我想和您聊聊我的职业规划"。

注意 2：用市场数据

不要说"我觉得我值 X"，说"据 XX 报告，类似岗位的市场价是 X"。

注意 3：不止谈钱

除了钱，还可以谈：

- 奖金。
- 股票 / 期权。
- 培训 / 学习机会。
- 工作时间灵活性。
- 团队 / 资源。

注意 4：不接受"会考虑"

老板说"会考虑"，你要主动约定"什么时候给您答复"。

注意 5：不在情绪里谈

不要在情绪激动时谈薪资，等冷静后再谈。

## 三、晋升谈判：5 步流程

### 步骤 1：理解晋升标准

不同公司有不同的晋升标准：

- KPI 完成率。
- 项目成果。
- 团队贡献。
- 能力提升。
- 同事评价。

你要先理解晋升标准，再针对性准备。

### 步骤 2：自评达标度

针对每个晋升标准：

- 我在哪些方面达到？
- 我在哪些方面部分达到？
- 我在哪些方面没达到？

### 步骤 3：补足差距

对没达标的方面，主动补足。

示例：

- 团队贡献不够，主动带项目。
- 同事评价不够，主动帮助同事。

### 步骤 4：呈现成果

用 4 步结构呈现：

- 立场："老板，我想和您聊聊晋升"。
- 支撑：3 个达标证据。
- 例子：1 个具体项目。
- 行动："下一步我希望晋升到 X 岗位"。

### 步骤 5：听反馈 + 调整

老板会反馈：

- 您在哪些方面达标？
- 您在哪些方面没达标？
- 您需要做哪些事才能晋升？

你针对反馈调整，3 个月后再谈。

### 晋升谈判的 3 个常见错误

错误 1：硬要

"我应该晋升了"，让老板防御。

错误 2：不听反馈

老板说"您还差点"，你不听，坚持要。

错误 3：短期思维

不为长期做铺垫，只想要短期晋升。

## 四、岗位调整：5 步流程

岗位调整是职场里常见的谈判，横向 / 纵向 / 转岗。

### 横向调整（同级别换岗）

适用：

- 你对当前岗位兴趣下降。
- 你想拓展能力。
- 团队变动需要你转岗。

### 纵向调整（晋升 / 降级）

纵向调整 = 晋升（参见上一节）或降级（被动）。

降级的谈判：

- 主动表达"我希望继续为公司贡献"。
- 主动表达"我接受 X 安排"。
- 不要公开抱怨。

### 转岗（换团队 / 换部门）

转岗流程：

1. 找目标部门负责人聊，了解需求。
2. 找当前老板聊，表达转岗意愿。
3. 内部协调（HR / 双方老板）。
4. 转岗 + 试用。

### 转岗谈判的 3 个注意

注意 1：不要公开抱怨

"我在这里没发展"，这种抱怨会让双方老板都不喜欢你。

正确："我想拓展能力 / 我对 X 方向更感兴趣"。

注意 2：先找目标再走

不要先离开再找，找好目标再走。

注意 3：保持和原老板关系

转岗后还要和原老板保持关系，职场很小。

## 五、工作资源争取：5 步流程

### 资源类型

职场里能争取的资源：

- 人（团队规模 / 招聘）。
- 钱（预算 / 设备）。
- 时间（deadline / 排期）。
- 关系（和谁合作）。
- 培训（学习机会）。

### 资源争取的 5 步

步骤 1：明确需求

我需要什么资源？为什么需要？

步骤 2：量化价值

这个资源能给公司带来什么？

步骤 3：方案化呈现

不要只说"我需要 X"，说"我打算用 X 做 Y，为公司带来 Z"。

步骤 4：谈

用本书所有策略。

步骤 5：兑现

资源到位后立刻行动，证明你能用好资源。

## 六、跨部门协调：5 步流程

跨部门协调是职场里的高频谈判。

### 跨部门协调的 5 个难点

难点 1：目标不同

每个部门有不同目标，跨部门协调要让目标对齐。

难点 2：优先级不同

每个部门有不同优先级，要协商共同优先级。

难点 3：资源竞争

每个部门争资源，要协商分配。

难点 4：沟通成本

跨部门沟通成本高，效率低。

难点 5：责任归属

跨部门项目责任归属不清，容易推诿。

### 跨部门协调的 5 步

步骤 1：理解对方

- 对方部门的目标是什么？
- 对方的痛点是什么？
- 对方的决策人是谁？

步骤 2：找共同点

- 双方有没有共同目标？
- 双方有没有共同痛点？
- 双方有没有共同利益？

步骤 3：方案化

- 不要用"我们要合作"，用"我们要做 X，达到 Y"。
- 用 4 步结构呈现。

步骤 4：建立机制

- 定期沟通（每周 1 次会议）。
- 明确责任（谁做什么）。
- 明确 KPI（共同 KPI）。

步骤 5：长期维护

- 关系维护（季度 / 半年 / 1 年）。
- 共同成果（季度复盘）。

## 七、职场谈判的常见错误

### 错误 1：被动等待

"老板应该主动给我加薪"，这种想法 90% 不会实现。

正确：主动谈判，主动争取。

### 错误 2：硬要

"我应该加薪了"，让老板防御。

正确：用方案 + 价值呈现。

### 错误 3：威胁

"不加薪我就走"，让老板觉得你不可靠。

正确：表达意愿，给老板选择空间。

### 错误 4：不准备

不准备，被老板问倒，失去机会。

正确：充分准备，量化结果 + 市场数据。

### 错误 5：情绪化

带着情绪谈，容易失控。

正确：先冷静再谈。

### 错误 6：不长期

不为长期做铺垫，短期谈判失败。

正确：长期主义，每次谈判都为未来铺路。
