# 第 8 章 开局策略

## 开篇：林涵的第一句话

林涵第一次见那个连锁零售集团的刘总，是在一个周五的下午。

她提前 10 分钟到，前台把她带到了 12 楼的会议室。刘总进来时她站起身，握手，微笑，递上名片。一切都做得不错。

刘总坐下，打开笔记本，看着她，等她开口。

林涵的脑子里突然一片空白。她原本准备的开场白是"非常感谢您抽时间……"，但她觉得这样太客气，可能会显得没内容。

她临时改口，脱口而出的是："刘总，我想先问一下，你们这个项目预算是多少？"

刘总的脸立刻沉了一下。他说："林总，我们先把您要介绍的内容介绍一下吧。"

林涵意识到，她第一句话就把整个会谈引到了错误的方向。后面 30 分钟无论她怎么调整，都很难回到"合作"的氛围。

后来她复盘：开场 5 分钟定下了接下来 90% 的走向。她那句"你们的预算是多少"，让对方立刻警觉"这个人是来卖东西的，不是来帮我们解决问题的"。

这一章我们就解决"开局 5 分钟"。

## 一、开局策略的 4 个目标

开局不只是"寒暄"。开局策略要同时达成 4 个目标。

### 目标 1：氛围

建立对方对你的初步印象，"这个人值得我花时间"。

错误做法：闲聊过多、问太直接的问题、过于正式。

正确做法：先聊人 3-5 分钟，建立轻松感，然后聊事。

### 目标 2：定位

让对方知道你是谁、你是来做什么的、你能给他什么。

错误做法：太谦虚（"我们公司很小"）、太骄傲（"我们是行业第一"）、太啰嗦（5 分钟自介）。

正确做法：30 秒内说清你是谁 + 你为什么在这里 + 你能给对方什么。

### 目标 3：信息

在前 5 分钟里获取对方的关键信息。

错误做法：急于问具体业务（林涵的错误）。

正确做法：先用开放式问题（"您能介绍一下您这个项目的背景吗？"）。

### 目标 4：议程

让对方接受你提出的谈判议程。

错误做法：对方主导议程、你没有议程。

正确做法：先提一个 5-7 项的议程，让对方同意。

这 4 个目标在开局前 5 分钟同时推进。任何 1 个目标失败，整个开局都失败。

## 二、第一印象塑造：3 招

### 招数 1：提前 5 分钟

早到 5 分钟，不是 30 分钟，不是 10 分钟。

早 30 分钟：显得你太紧张。

晚 10 分钟：显得你不尊重对方。

早 5 分钟：刚刚好，你到了，对方还没到，你能熟悉环境、调整心态。

### 招数 2：微笑弧度

微笑时让眼角也参与，这是"真笑"。

怎么练：每天对镜练习 5 分钟，让真笑自然出现。

实战建议：

- 见到对方的 3 秒内，给一个真笑。
- 接下来 30 秒内，慢慢过渡到正常表情。

### 招数 3：命名效应

让对方从第一次接触就记住你的名字。

技巧：

- 自介时说慢一点、清晰一点："我叫林涵，林是森林的林，涵是涵养的涵"。
- 把名字和对方的关联点说出来："我和您的刘总名字同音，但我是 3 水的涵"。
- 让对方说出你的名字（在某次回应时让对方叫你的名字，强化记忆）。

实战建议：

- 第一次见面要让对方至少叫一次你的名字。
- 你也要叫对方的名字至少 3 次。

## 三、建立合作氛围：3 种方式

### 方式 1：先聊人，再聊事

开局 3-5 分钟，聊 3 个轻松的话题：

- 对方公司附近的环境（"这个园区挺安静"）。
- 共同认识的人（"李总上次提到您"）。
- 共同关心的事（"最近这个行业的展会您看了吗？"）。

实战建议：

- 不要聊得太深（不要聊家庭 / 政治 / 宗教）。
- 不要超过 5 分钟。

### 方式 2：共同敌人 / 共同利益

找一个"共同的敌人"或"共同的愿望"。

共同敌人："这个行业里大家都被 X 问题困扰"。

共同利益："我们都在推动 Y 方向"。

实战建议：

- 共同敌人比共同利益更有效，共同敌人有情感共鸣。
- 不要把客户说成"共同的敌人"，这是商业大忌。

### 方式 3：先聊合作的可能性

不要说"我是来卖东西的"，说"我们一起看看有没有合作的可能性"。

话术对比：

- 错误："我想给您介绍一下我们的产品"，对方立刻进入"评估产品"模式。
- 正确："我想和您聊聊您最近面临的挑战，看我们是不是能帮上忙"，对方进入"分享挑战"模式。

实战案例：

- 林涵后来学了一个开场："刘总，我最近看了你们公司在零售数字化方面的几个新闻，我特别想知道，你们现在最大的挑战是什么"。
- 客户立刻开始分享他的问题。
- 林涵接下来 10 分钟都在问问题，没有推销。

## 四、率先报价：先报价的 4 个好处和 3 个风险

### 先报价的 4 个好处

好处 1：设锚

你的第一报价成为对方判断"合理价位"的参考。

好处 2：占主动

先报价意味着你先在博弈里占据主动。

好处 3：展示信息

你的报价透露了你的成本 / 期望 / 紧迫度。

好处 4：加速谈判

对方如果接受或拒绝，你的谈判时间缩短。

### 先报价的 3 个风险

风险 1：暴露底牌

你的报价透露了你的真实价位。

风险 2：对方立刻拒绝

如果对方早有更低的目标，先报价等于自杀。

风险 3：缺少调整空间

一旦先报价，你的让步空间被限定。

### 决策规则：什么时候先报价？

| 你的情境 | 先报价 | 不先报价 |
|---------|--------|---------|
| 你有信息优势（你懂市场 / 你懂成本） | 是 | |
| 对方是新手 / 不熟悉行情 | 是 | |
| 你是大公司 / 你有权威 | 是 | |
| 你处于劣势（你需要这单） | | 是 |
| 你不知道对方的预算 / 你不知道对方的真实需求 | | 是 |
| 对方是专家 / 对方有强势地位 | | 是 |

## 五、如何回应首次报价：4 步法

对方报价后，你不要立刻回应。用 4 步法：

### 步骤 1：听清

让对方说完。即使对方的报价让你惊讶，也不要立刻反应。

实战："我听您说一下……"。

### 步骤 2：复述

把对方的报价用自己的话复述一遍。

实战："所以您这边希望整体报价控制在 280 万以内，对吗？"。

复述的好处：

- 让对方知道你在听。
- 让对方有机会修正。
- 让整个房间的人对"对方的报价"形成一致理解。

### 步骤 3：确认范围

除了数字，确认其他范围。

实战：

- "除了价格，您对付款方式 / 交付时间 / 售后服务有什么具体要求吗？"
- "您这边的预算是只包括 X 部分，还是包括全部？"

实战技巧：

- 用"5W2H"逐项确认。

### 步骤 4：引导下一步

不要让谈判停留在"报价 - 复述"。要引导下一步。

实战话术：

- "好，我理解您的报价。让我先回去对一下我们的成本，明天给您具体答复，可以吗？"
- "我有一个想法，我们先不直接谈价格，先看看方案能不能解决您的问题。如果方案 OK，我们再谈价格。"

实战技巧：

- 给谈判一个"缓冲期"。
- 把"现在就拍板"改成"先讨论方案"。

## 六、5 类开局话术

### 话术 1：问候型

特征：以"您好""感谢""很高兴"开头。

示例：

- "非常感谢您抽时间见面，今天聊 30 分钟可以吗？"
- "很高兴见到您，您比照片上更年轻。"

适用：

- 第一次见面。
- 文化场景较为正式。

### 话术 2：共鸣型

特征：先认同对方的某种处境。

示例：

- "我听说您最近在拓展华东市场，我们也一直在关注这个方向。"
- "您之前提到的供应链问题，我们这边也有客户面临类似挑战。"

适用：

- 你已经了解对方。
- 你希望快速建立连接。

### 话术 3：提问型

特征：用一个开放式问题开场。

示例：

- "今天我特别想了解您这个项目的核心挑战是什么。"
- "您能介绍一下您对这个项目的整体思路吗？"

适用：

- 你不知道对方的需求。
- 你希望对方主动分享。

### 话术 4：数据型

特征：以一个数据 / 案例开场。

示例：

- "我最近看了艾瑞咨询的报告，里面提到零售数字化的几个关键趋势，其中 3 个和您目前的业务高度相关。"
- "我们服务的一家类似连锁集团（暂不说名字），去年做到了 X 业绩提升。"

适用：

- 你希望用事实开场。
- 对方是数据型决策者。

### 话术 5：故事型

特征：以一个具体故事开场。

示例：

- "我上周去了 XX 客户那里，他们做了一个让您觉得不可思议的事情，他们把库存周转率提升了 30%。我今天想和您分享他们的做法。"

适用：

- 你希望引起对方兴趣。
- 对方是故事型 / 视觉型思考者。

实战建议：

- 5 类话术不要都用，选 1 类。
- 第一次见面用"问候型 + 提问型"组合最安全。
- 对方是熟人 / 关系好时用"共鸣型 + 故事型"。

## 七、5 类开局雷区

### 雷区 1：暴露底线

"我们的底价是 340 万"，你的底线被对方知道了。

纠正：永远不要在开局时透露你的底线。

### 雷区 2：强求承诺

"您今天能不能给个答复？"，对方会觉得被逼。

纠正：把"答复"改成"继续"，"那我们下一步看看方案是不是合适"。

### 雷区 3：立刻让步

"您觉得价格不合适？我们降 10% 看看"，你的让步没有价值。

纠正：让步要在对方的让步之后；让步要"换"对方的让步。

### 雷区 4：长篇大论

开局 5 分钟你一直说，对方没有机会开口，对方不会同意。

纠正：50% 时间说，50% 时间听。

### 雷区 5：自吹自擂

"我们公司是行业第一"，对方会警惕。

纠正：用具体案例替代自吹，"我们服务过 XX 客户，他们目前做到了 Y 业绩"。

## 八、一个正面案例：陈伟的开局挽回

陈伟在德国那次失败后，重新设计了所有大客户的开局策略。

他第一次见美国一家上市公司 CEO 时，新的开局是这样的：

1. 提前 5 分钟到（招数 1）。
2. 握手 + 真笑 + 自介时强调共同朋友（"David 提到您很多次"）。
3. 聊 3 分钟行业（共鸣型话术）："我最近看了您公司上个月发布的财报，里面那个新方向我觉得很有意思……"。
4. 引导议程："今天我准备了 4 个话题，您过去 12 个月的最大挑战、我们的方案如何帮您解决、我们和其他客户合作的案例、下一步。30 分钟够吗？"。

这次开局让 CEO 一开始就感觉"这个人懂我们"，会谈结束时他们签订了合作意向书。

### 这个案例的关键：

- 提前到（细节）。
- 共鸣开场（连接）。
- 议程引导（结构）。
- 时间框定（边界）。

## 九、一个反面案例：苏清的开局把老板逼到对立面

苏清想跟老板争取新项目预算。她准备了一个 30 分钟的会议。

她进入老板办公室坐下后，第一句话是："老板，我想争取 50 万的预算。"

老板立刻问："为什么是 50 万？"，对话进入了"论证 50 万合理性"的模式。

苏清接下来 20 分钟都在论证为什么 50 万是合理的。但老板心里在想："这 50 万要不要给？"

30 分钟结束时，老板说："我再考虑考虑。"

苏清失败的原因，她一开局就暴露了诉求（要 50 万）。她应该一开局先聊"老板您今年的 KPI 是什么"，找到老板真正的痛点，再把 50 万作为"解决老板痛点的方案"。

### 这个案例的反面教训：

- 开局不要直接说诉求。
- 开局要先听对方的需求。
- 然后把你的诉求包装成"解决对方需求"的方案。

## 十、4 类开局组合

按"对手 + 情境"选开局策略：

### 组合 1：合作型开局

适合：长期合作伙伴 / 关系好的客户 / 简单续约。

话术：

- 问候 + 共鸣。
- 议程简短（3 项以内）。
- 时间 15-30 分钟。

### 组合 2：竞争型开局

适合：强势客户 / 价格战 / 招标。

话术：

- 问候 + 数据（用数据立威）。
- 议程完整（5-7 项）。
- 时间 30-60 分钟。

### 组合 3：信息型开局

适合：第一次见面 / 不熟悉对方 / 复杂合作。

话术：

- 问候 + 提问。
- 议程开放（根据对方回答调整）。
- 时间 30-45 分钟。

### 组合 4：试探型开局

适合：跨文化 / 跨行业 / 对方背景未知。

话术：

- 问候 + 故事（用一个对方可能感兴趣的故事）。
- 议程保守（3 项）。
- 时间 15-30 分钟。

## 章末行动清单

下次谈判 / 会议前，提醒自己：

1. 提前 5 分钟到。
2. 微笑时让眼角也参与。
3. 30 秒自介说清我是谁 + 我为什么在这里 + 我能给什么。
4. 聊 3 分钟人再聊事。
5. 用 1 个开放式问题开场。
6. 提出 5-7 项议程。
7. 永远不暴露底线。
8. 不立刻让步。

## 实战演练

**演练 1**：写下你下一次谈判的开局前三句话。

要求：

- 第一句：问候或共鸣。
- 第二句：30 秒自介。
- 第三句：开放式问题。

演练方式：对镜说 3 遍，录音回放。

**演练 2**：设计你下一次谈判的议程。

要求：

- 议程 5-7 项。
- 每项分配时间。
- 议程前 1-2 项是"暖场 + 信息收集"。
- 议程后 1-2 项是"收口 + 下一步"。

**演练 3**：录一段你和别人 30 分钟对话的开头（前 5 分钟）。回看 1 遍：

- 你说话占比多少？
- 你有没有透露底线？
- 你有没有用 1 个开放式问题？
- 你有没有建立合作氛围？
