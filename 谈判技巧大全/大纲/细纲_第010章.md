# 细纲：第 10 章 控制谈判节奏

## 一句话核心事件

读者读完这一章，能主动控制谈判节奏而不是被拖着走，能在合适的时候"暂停 / 加速 / 减速"。

## 章首钩子（案例切入）

> "苏清被老板拉去开一个'紧急预算会议'，老板越说越快，苏清一句话都插不上。会议结束时她被'会签'了 50 万的预算。回家路上她意识到——她不是在'谈判'，她是在'被谈判'。"

## 子情节点序列（约 12 个情节点）

1. 苏清"被谈判"的失败案例
2. 节奏控制的 3 类：快 / 慢 / 暂停
3. 时间压力：对方用时间压力时怎么办（识别 / 验证 / 反弹 / 退出）
4. 节奏调整：5 类节奏切换（从快到慢 / 从慢到快 / 从无到有 / 从有到无 / 维持稳定）
5. 暂停与休会：3 个使用时机 + 3 步收尾
6. 分阶段推进：5 阶段（开局 / 探测 / 谈判 / 收尾 / 维护）
7. 一个正面案例：陈伟怎么在客户催单时主动"暂停"扭转局面
8. 一个反面案例：林涵怎么在客户"我们赶时间"压力下让步 20%
9. 节奏工具：日历 / 倒计时 / 议程表 / 缓冲日 / 24 小时回话规则
10. 节奏与情绪：情绪紧张时容易跑快，需要刻意"踩刹车"
11. 跨文化节奏差异：美国人快 / 法国人慢 / 阿拉伯人多停顿 / 中国人看关系
12. 章末行动清单 + 实战演练（为你下一次谈判设计一份"节奏方案"）

## 章首/章尾钩子

- 章首：被时间压力拖走的失败案例
- 章尾："请现在为你下一次谈判设计一份'节奏方案'——快 / 慢 / 暂停 怎么安排"

## 目标读者收获

1. 能用 3 类节奏切换应对不同局面
2. 能用 5 类节奏工具
3. 能在 5 阶段里识别当前在哪个阶段
4. 能用 3 步收尾一次暂停
5. 能在跨文化场景下调整节奏
