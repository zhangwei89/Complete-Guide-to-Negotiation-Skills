# 细纲：第 2 章 谈判心理学

## 一句话核心事件

读者读完这一章，会知道至少 5 种"心理机制"如何系统在谈判中起作用，并能在下一次谈判前对自己说一句"我现在犯的是哪一条"。

## 章首钩子（案例切入）

> "陈伟第一次去见投资人，对方开口就说：'你这估值太高了，我们都是按你 60% 的估值来谈。'陈伟心里咯噔一下——他不知道这叫'锚定效应'。他本能地把估值往下调了一档，回到家后悔了半年。"

## 子情节点序列（约 12 个情节点）

1. 陈伟第一次见投资人的案例切入（投资人开低价锚定 → 陈伟被动应价 → 后面想反悔但已签字）
2. 利益与立场（Fisher-Ury 框架）：为什么立场之争会把蛋糕做小
3. 人性的决策规律：损失厌恶 / 锚定 / 框架效应 / 现状偏好
4. 情绪如何影响谈判：愤怒 / 焦虑 / 兴奋 / 恐惧 的生理反应
5. 信任建立机制：能力信任 / 善意信任 / 透明信任 三层
6. 心理预期管理：上推 / 下拉 / 锚定 三招
7. 西奥迪尼的影响力 6 原理：互惠 / 承诺 / 社会认同 / 喜好 / 权威 / 稀缺
8. 一个反面案例：陈伟如何被"互惠"原理套路（对方送了一本书让他不好意思拒绝）
9. 一个正面案例：苏清如何用"承诺一致性"原理争取晋升（让老板先说出一句"你很优秀"）
10. 心理学的边界：人是理性 + 情绪的混合体，关键是学会在两种状态间切换
11. 心理博弈的元能力：能识别自己正在被哪个原理影响
12. 章末行动清单 + 实战演练（识别最近一次谈判里你用了哪条心理原理）

## 章首/章尾钩子

- 章首：失败案例切入（被锚定 → 失血）
- 章尾："现在请你回想最近一次谈判，对方用了哪个心理原理套路你？"

## 目标读者收获

1. 能区分"立场"和"利益"——并在谈话中说出利益而不是立场
2. 能识别 6 种心理原理
3. 能识别自己情绪被带动的 4 个身体信号
4. 能在 3 步内说出对方在用哪种心理原理影响你

5000 字（带心理学较多内容）
