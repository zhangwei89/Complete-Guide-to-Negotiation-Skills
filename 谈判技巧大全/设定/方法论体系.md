# 方法论体系（理论-工具-策略-场景-案例-训练）

本书有一条贯穿 33 章的方法论主线，叫"六层结构"。它不是装饰，而是一套让"读完就能用"的内容设计框架。附录里的清单、话术、模板也是这套结构的延伸。

## 六层之间的关系

```
        ┌──────────── 训练 ────────────┐
        │                              │
     ┌─案例─┐                       ┌─反馈─┐
     │      │                       │      │
  ┌─场景─┐ ┌─策略─┐ ┌─工具─┐ ┌─理论─┐
  │      │ │      │ │      │ │      │
  └──┬───┘ └──┬───┘ └──┬───┘ └──┬───┘
     │        │        │        │
   场景化   策略化   工具化   原理化
     │        │        │        │
     ▼        ▼        ▼        ▼
       让你在真实场景里能用起来
```

## 每一层的具体含义

### 第一层：理论（Why）

回答"为什么这么做有效"。本书援引的核心理论源流有六条：

1. 哈佛谈判项目（Harvard Negotiation Project）—— Fisher、Ury 的"原则谈判"，强调利益与立场的区分。
2. 罗杰·费希尔与威廉·尤里《谈判力》（Getting to Yes）—— BATNA、客观标准、利益、选项四大支柱。
3. 罗伯特·西奥迪尼《影响力》（Influence）—— 互惠、承诺与一致、社会认同、喜好、权威、稀缺六大说服原理。
4. 理查德·谢尔（Richard Shell）—— 谈判的"风格五型"（避免型、迁就型、竞争型、合作型、妥协型）。
5. 阿克塞尔罗德（Robert Axelrod）《合作的进化》—— 重复博弈与"以牙还牙"策略。
6. 卡尼曼与特沃斯基（Kahneman & Tversky）—— 前景理论与损失厌恶。

### 第二层：工具（What）

可以直接拿来用的清单、表格、模板。本书原创的工具包括：

- BATNA 评估表
- 目标区间表（理想值、可接受值、底线）
- 利益分析矩阵（对方明显利益 / 对方隐藏利益 / 我方明显利益 / 我方隐藏利益）
- 风险评估表（出问题的概率 × 影响的乘积）
- 谈判准备 Checklist
- 谈判复盘模板（见附录 D）

### 第三层：策略（How to plan）

在不同情境下选用的策略组合。经典策略包括：

- 锚定与反锚定
- 高开低走 vs 低开高走
- 等价交换（让步换让步）
- 时间压力施加与化解
- 沉默与休会
- 红脸白脸与黑白配
- 条件打包
- BATNA 公开与隐藏

策略不是孤立招式，而是组合拳。同一套策略在两个不同场景里走出来的结果完全不同。

### 第四层：场景（Where）

九大场景：商务、职场、销售、采购、消费、家庭、国际、团队、危机。每个场景的约束条件不一样，所以策略不能照搬。

比如 BATNA 在商务谈判里是杀手锏，但在家庭对话里如果抛出"BATNA"，气氛会立刻僵硬化。

### 第五层：案例（Proof）

两类案例：

1. **化名案例**：来自真实经历的转译，把人名、公司、品牌换成虚拟的，但场景、金额、时间线保留现实感。
2. **经典公开案例**：已经过媒体广泛报道、读者可以在网上核实真相的案例（例如美苏 1962 古巴导弹危机的"肯尼迪-多勃雷宁"会谈，或某个公开的并购案例）。

每章正文至少有 2 个案例：1 个成功、1 个失败或踩坑。

### 第六层：训练（Practice）

让"知道"变成"做到"。每章末尾保留两类内容：

1. **行动清单**：3-7 条可勾选项，读者可以打印贴在办公桌前。
2. **实战演练**：1-2 个可立刻执行的小练习，可能是一段对话、一次自我反思、一份清单填写。

## 为什么是这套结构

- 理论章先建立心智地图，让读者知道"为什么"。
- 沟通章把心智地图转成日常对话能力。
- 策略章进入"用什么招"。
- 心理章解释"为什么会赢会输"。
- 场景章把方法落到具体场景里。
- 案例章让所有方法变成可复盘的故事。
- 训练章把"知道"逼到"做到"。

读者至少可按两种姿势读：

1. **从头到尾顺序读**：建立完整的谈判操作系统。
2. **按需跳读**：遇到具体谈判场景时翻到对应章节。
